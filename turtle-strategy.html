核心概念解析

海龟交易策略 — 核心思想与起源

海龟交易策略源于1980年代，由商品交易大师理查德·丹尼斯设计。他相信交易可以被传授，如同培养"海龟"一样。策略核心是趋势跟踪：在价格突破近期高点时入场，在跌破近期低点时离场，配合ATR自适应止损，实现"截断亏损，让利润奔跑"。

关键优势：①趋势跟踪，自动适应市场方向；②ATR自适应止损，波动大时止损宽、波动小时止损紧；③规则明确无歧义，消除情绪干扰；④分散化应用，可同时覆盖多品种；⑤正期望系统，小亏大盈。

唐奇安通道 (Donchian Channel) — 高低点通道

唐奇安通道由 Richard Donchian 发明，是海龟策略的入场和出场核心。通道由过去N日的最高价和最低价构成：

上轨 = max(High, 前 N 日)
下轨 = min(Low, 前 N 日)

入场规则：当价格突破入场上轨（20日最高价）→ 买入。这意味着市场创出近期新高，可能开启上升趋势。

出场规则：当价格跌破出场下轨（10日最低价）→ 卖出。这意味着市场创出近期新低，趋势可能结束。

注意：通道使用前N日数据（不含当日），避免未来函数。入场周期(20) > 出场周期(10)，使得入场更谨慎、出场更灵敏。

平均真实波幅 (ATR) — 衡量市场波动

ATR 由 J. Welles Wilder 发明，衡量市场波动程度。先计算每日的真实波幅(TR)：

TR = max( High - Low, |High - 前收盘|, |Low - 前收盘| )
ATR = TR 的 N 日移动平均

TR 取三者最大值，考虑了跳空缺口的影响。ATR 越大，市场波动越剧烈；ATR 越小，市场越平静。

在海龟策略中的作用：ATR 用于计算止损价位和仓位大小。波动大的股票止损宽一些（避免被噪声震出），波动小的股票止损紧一些（控制亏损幅度）。

2N 止损法则 — 自适应止损

海龟策略的止损规则：买入后立即设定止损价 = 买入价 - 2 × ATR。

止损价 = 买入价 - 2 × ATR
亏损额度 ≈ 2 × ATR（约2个波动单位）

为什么是2倍ATR？①太窄（1N）容易被正常波动震出；②太宽（3N+）单笔亏损过大；③2N在统计上覆盖约95%的正常波动范围，只有真正趋势反转才会触发。

止损是海龟策略最重要的风控机制。每笔交易的最大亏损被锁定在约2N，而盈利交易可以远远超过这个数字，从而实现"截断亏损，让利润奔跑"的正期望系统。

最大回撤 (Maximum Drawdown, MDD)

策略净值从历史最高点回落到后续最低点的最大幅度，衡量策略在最坏情况下的亏损程度。

MDD = min( (NAV(t) - max(NAV(1..t))) / max(NAV(1..t)) ) × 100%

MDD < 10% 属低风险，10%~20% 中等风险，> 30% 风险较高。投资者应确保 MDD 在自身心理承受范围内。

夏普比率 (Sharpe Ratio)

衡量每承担一单位总风险所获得的超额回报，由诺贝尔奖得主威廉·夏普提出。

Sharpe = √252 × (R_mean - R_f) / R_std

Sharpe > 1 策略优秀，> 2 极好，< 0 意味着不如无风险利率。它是评估策略风险调整后收益的核心指标。

累计回报 (Cumulative Return)

策略从期初到期末的总收益率，反映绝对盈利能力。

Cumulative Return = (NAV_final - 1) × 100%

需结合MDD和夏普比率综合评估。高回报伴随高回撤的策略，实际投资体验可能不如回报稍低但回撤可控的策略。
